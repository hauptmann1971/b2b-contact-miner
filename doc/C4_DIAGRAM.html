Level 3

Component Diagram - Main Pipeline

Purpose: Shows how the main pipeline is broken down into components.

📥 Input Layer

  • Keyword Loader - Loads keywords from database
  • State Manager - Tracks pipeline progress
↓

🔍 Search Layer

  • Search Orchestrator - Coordinates search workflow
  • SERP Getter - Fetches results from SERP API
  • Result Processor - Parses and validates results
↓

🕷️ Crawl Layer

  • Domain Crawler - Crawls websites with Playwright
  • Content Extractor - Extracts HTML content
  • Rate Limiter - Prevents overwhelming servers
↓

🤖 AI Extraction Layer

  • LLM Caller - Calls YandexGPT/GigaChat
  • Contact Parser - Parses LLM JSON response
  • Validation - Validates extracted contacts
↓

💾 Storage Layer

  • Result Saver - Saves to database
  • Export Service - Exports to CSV/Excel
  • Task Queue Writer - Creates async tasks
