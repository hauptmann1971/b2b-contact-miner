# 🚀 Быстрый старт Flask Web Server

## Установка зависимостей

Если зависимости еще не установлены, выполните:

```bash
py -m pip install -r requirements.txt
```

Или установите только Flask:

```bash
py -m pip install flask
```

## Запуск веб-сервера

```bash
py web_server.py
```

Сервер запустится на **http://localhost:5000**

## Что вы увидите

### Главная страница
- Статистика системы (ключевые слова, домены, контакты)
- Форма для добавления новых ключевых слов
- Таблица последних ключевых слов
- Навигация по разделам

### Health Check эндпоинты
- `/health` - полная проверка здоровья
- `/health/live` - проверка живости
- `/health/ready` - проверка готовности

### API эндпоинты
- `/api/stats` - статистика в JSON
- `/api/keywords` - список ключевых слов в JSON

## Возможные проблемы

### ModuleNotFoundError
Если видите ошибку импорта модулей, установите зависимости:
```bash
py -m pip install pydantic-settings sqlalchemy pymysql cryptography
```

### База данных недоступна
Убедитесь, что MySQL запущен и настройки в `.env` корректны.

### Порт 5000 занят
Измените порт в конце файла `web_server.py`:
```python
app.run(host='0.0.0.0', port=8080, debug=True)
```

## Использование

1. Откройте браузер и перейдите на http://localhost:5000
2. Добавьте ключевые слова через форму
3. Просматривайте результаты в разделах "Ключевые слова" и "Контакты"
4. Используйте health check эндпоинты для мониторинга
