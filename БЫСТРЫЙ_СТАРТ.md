# 🚀 B2B Contact Miner - Быстрый старт

## ⚡ Минимальные шаги для запуска

### 1️⃣ Настройка (один раз)

```bash
# Скопировать конфигурацию
copy .env.example .env          # Windows
cp .env.example .env            # Linux/Mac

# Отредактировать .env (указать DATABASE_URL и API ключи)

# Установить зависимости
pip install -r requirements.txt

# Проверить готовность
python validate_setup.py
```

### 2️⃣ Запуск

```bash
# Windows (CMD)
start_all.bat start

# Windows (PowerShell)
.\start_all.ps1 start

# Linux/Mac
./start_all.sh start
```

### 3️⃣ Использование

Откройте в браузере:
- **Главная**: http://localhost:5000
- **Контакты**: http://localhost:5000/contacts
- **Здоровье системы**: http://localhost:8000/health

### 4️⃣ Запуск pipeline

```bash
# Автоматически (ежедневно в 02:00)
# Планировщик запущен вместе с сервисами

# Или вручную
python main.py
```

## 📋 Основные команды

```bash
# Управление сервисами
./start_all.sh start      # Запустить все
./start_all.sh stop       # Остановить все
./start_all.sh restart    # Перезапустить
./start_all.sh status     # Проверить статус

# Логи
./start_all.sh logs       # Все логи
./start_all.sh logs web_server   # Логи конкретного сервиса

# Очистка
./start_all.sh clean      # Удалить старые PID и логи
```

## 🔍 Проверка готовности

```bash
python validate_setup.py
```

Эта команда проверит:
- ✅ Python версии 3.8+
- ✅ Наличие файла .env
- ✅ Установку всех зависимостей
- ✅ Подключение к базе данных
- ✅ Доступность портов 5000 и 8000

## 🆘 Проблемы?

### Сервисы не запускаются
```bash
# 1. Проверить среду
python validate_setup.py

# 2. Посмотреть логи
./start_all.sh logs

# 3. Очистить и перезапустить
./start_all.sh stop
./start_all.sh clean
./start_all.sh start
```

### Порт уже занят
```bash
# Найти процесс
lsof -i :5000              # Linux/Mac
netstat -ano | findstr :5000   # Windows

# Убить процесс
kill <PID>                 # Linux/Mac
taskkill /PID <PID> /F     # Windows
```

### База данных не подключается
```sql
-- Создать базу данных
CREATE DATABASE contact_miner;

-- Проверить подключение
mysql -u user -p contact_miner
```
